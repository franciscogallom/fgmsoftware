Responsive web design.

El diseño web adaptable es una filosofía de diseño y desarrollo cuyo objetivo es adaptar la apariencia de las páginas web al dispositivo que se esté utilizando para visitarlas. Hoy día las páginas web se ven en multitud de dispositivos como tablets, smartphones, libros electrónicos, portátiles, PC, etcétera. Esta tecnología pretende que con un único diseño web, todo se vea correctamente en cualquier dispositivo.
